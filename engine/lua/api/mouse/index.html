Mouse

Modul obsahuje funkce pro obsluhu myši (jasně že počítačové, rozhodně vám neumožní podrobit si krysí národ podobně jako v Dishonored).

Metody

button_index

function button_index(button_name)

Vrátí index tlačítka který má název button_name.

Parametr Typ Popis
button_name str Název tlačítka

button_name

function button_name(button_index)

Vrátí název tlačítka nebo prázdný řetezec pokud je index špatný.

Parametr Typ Popis
button_index int Index tlačítka

button_state

function button_state(button_index)

Vrátí stav tlačítka v aktuálním frejmu.

Parametr Typ Popis
button_index int Index tlačítka

pressed

function pressed(button_index)

Vráti true pokud nebylo tlačítko v minulém frejmu stisknuté ale v aktuálním je jinak vrací false.

Parametr Typ Popis
button_index int Index tlačítka

released

function released(button_index)

Vráti true pokud bylo tlačítko v minulém frejmu stisknuté ale v aktuálním není jinak vrací false.

Parametr Typ Popis
button_index int Index tlačítka

axis (button_index) : Vector2

Return mouse position
