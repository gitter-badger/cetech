Package

Modul obsahuje funkce pro obsluhu resource package.

Metody

create

function create(package_name)

Vytvoří id balíčku podle package_name.

Parametr Typ Popis
package_name str Název balíčku

load

function load(package_id)

Načte všechny resource které jsou v balíčku v případě že resource je již načtený nebude se znovu načítat ale zvýší se počítadlo referenci. Data nejsou hned dostupná ale nahrávají se na pozadí.

Parametr Typ Popis
package_id stringid64 Id balíčku

unload

function unload(package_id)

Pro všechny resource které jsou v balíčku sníží počítadlo a pokud resource nikdo nepoužívá smaže ho z paměti.

Parametr Typ Popis
package_id stringid64 Id balíčku

is_loaded

function is_loaded(package_id)

Vrátí true pokud jsou všechny resource které jsou v balíčku načteny v paměti a připraveny k používaní jinak vrací false.

Parametr Typ Popis
package_id stringid64 Id balíčku

flush

function flush(package_id)

Počká dokud nejsou načteny všechny resource které jsou v balíčku.

Parametr Typ Popis
package_id stringid64 Id balíčku
